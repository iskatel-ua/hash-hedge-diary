Install guide

  1. Download archive hash-hedge-chrome.zip using the Download button above.
  2. Extract the archive and open chrome://extensions in Chrome.
  3. Enable Developer mode and click Load unpacked.
  4. Select the extracted folder that contains manifest.json.
  1. Download archive hash-hedge-firefox.zip using the Download button above.
  2. Open about:debugging in Firefox address bar.
  3. Click "This Firefox" in the left sidebar.
  4. Click "Load Temporary Add-on" and select manifest.json from the extracted folder.
  5. For permanent installation, see detailed guide.

What the extension does

Hash Hedge Diary automatically collects your closed trade history from Hash Hedge, applies smart filters, and calculates key performance metrics including win rate, ROI, recovery factor, average hold time, and profit consistency. The popup gives you a fast performance snapshot so you can review your decisions and improve your trading process over time.
